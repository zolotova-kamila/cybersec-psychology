# Анализ конкурентов — 11 марта 2026

## Общая ситуация на рынке

### 1. Психологи общей практики (конкуренты низкого уровня)
**Примеры:** Психологи на Яндекс.Услугах, Профи.ру
- Предлагают общую терапию без специализации
- Цены: 2000-4000₽ за сессию
- **Недостатки:** Не понимают специфику IT и ИБ
- **Угроза:** Низкая — отсутствие экспертности в нише

### 2. Психологи с узкой специализацией (прямые конкуренты)
**Примеры:** Психологи для разработчиков, коучи по выгоранию
- Работают с IT-сферой, но без фокуса на кибербезопасности
- Цены: 4000-7000₽ за сессию
- **Преимущества:** Уже работают с техническими специалистами
- **Угроза:** Средняя — возможность расширения специализации

### 3. Корпоративные программы EAP (высокий уровень)
**Примеры:** Сервисы корпоративной психологической помощи
- Предлагают пакеты для компаний
- Цены: 50 000-200 000₽ в месяц
- **Недостатки:** Общий подход, нет специфики ИБ
- **Угроза:** Средняя — при наличии бюджета компании предпочтут комплексное решение

## SEO-конкуренты (топ-10 по ключевым запросам)

### Запрос: «психолог кибербезопасность»
1. Статьи на Хабре о психологии ИБ
2. Блоги компаний-разработчиков (Касперский, etc.)
3. Научные публикации

### Запрос: «психолог КПТ онлайн»
- Высокая конкуренция (Яндекс.Справник, Профи.ру)
- Требуется локальная SEO-оптимизация

### Запрос: «зашита от мошенников психология»
- Низкая конкуренция
- Возможность занять нишу

## Уникальное торговое предложение (УТП) Камилы Золотовой

### Что отличает от конкурентов:
1. **Единственный психолог КПТ с фокусом на кибербезопасность**
2. **Понимание специфики SOC и alert fatigue**
3. **Постинцидентная поддержка — редкая услуга**
4. **Комбинация КПТ + EMDR для работы с травмой**

### Рекомендации по позиционированию:
1. Акцент на уникальности ниши
2. Контент-маркетинг через профильные каналы (Хабр, Инфосек)
3. Корпоративное направление — работа с SOC-командами
4. Выступления на конференциях по ИБ

## Идеи контента на основе анализа:
1. Сравнение: «Чем отличается работа с психологом-«технарём» от обычного психолога»
2. Кейс-стади: «Как команда SOC восстановилась после инцидента»
3. Интервью с CISO о психологии безопасности
