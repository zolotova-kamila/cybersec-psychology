## 🔧 Техническая реализация

### Псевдокод фильтрации:
```javascript
const allowedKeywords = [
  'записаться', 'консультация', '/consultation',
  'вопросы', 'faq', '/faq',
  'материалы', '/materials',
  'цены', 'стоимость', '/price',
  'контакты', '/contact',
  '/start', 'меню', 'помощь'
];

function handleMessage(message) {
  const text = message.toLowerCase();
  
  // Проверяем, содержит ли сообщение разрешённые ключевые слова
  const isAllowed = allowedKeywords.some(keyword => text.includes(keyword));
  
  if (isAllowed) {
    // Обрабатываем команду
    processCommand(text);
  } else {
    // Отправляем стандартный ответ
    sendMessage("Извините, не понял ваш вопрос. Передам его специалисту.");
    showMainMenu();
  }
}
```

### Ответ на неразрешённые сообщения:
**Текст:**
```
Извините, не понял ваш вопрос. Передам его специалисту.

Вы можете:
• 📝 Оставить заявку на консультацию
• ❓ Посмотреть частые вопросы
• 📚 Получить бесплатные материалы
• 💰 Узнать стоимость
```

**Кнопки:**
- 📝 Записаться
- ❓ FAQ
- 📚 Материалы
- 💰 Цены

## 📊 Статистика

Каждое непонятное сообщение:
1. Фиксируется в логах
2. Отправляется специалисту (Камиле) для анализа
3. Клиенту показывается меню с кнопками

---

**Важно:** Бот НЕ ведёт терапевтические беседы и НЕ даёт психологических советов. Он только:
- Принимает заявки
- Отвечает на FAQ
- Отправляет материалы
- Перенаправляет сложные вопросы специалисту
