# Корпоративный пакет «Антистресс-карта»

## Концепция
Продукт для HR и руководителей IT-команд: диагностика + меры профилактики выгорания и тревожности у сотрудников.

---

## Варианты пакетов

### Пакет «Базовый» (для малых команд 5–15 человек)
**Цена:** 25,000₽
**Что входит:**
1. Онлайн-опрос команды (15 мин) — выявление уровня стресса
2. Индивидуальный отчёт для руководителя (PDF, 3–5 страниц)
3. 1 групповая сессия (90 мин, Zoom) — техники снижения тревожности
4. Чек-лист «3 сигнала выгорания» для руководителей

**Формат:** Онлайн
**Срок:** 1 неделя

---

### Пакет «Стандарт» (для средних команд 15–50 человек)
**Цена:** 55,000₽
**Что входит:**
1. Полный аудит команды (опросы + 3 интервью с ключевыми сотрудниками)
2. Детальный отчёт с рекомендациями (PDF, 10–15 страниц)
3. 3 групповые сессии (по 90 мин):
   - Сессия 1: Распознавание стресса
   - Сессия 2: Техники саморегуляции
   - Сессия 3: Коммуникация в кризисе
4. Памятка для сотрудников (PDF, красивый дизайн)
5. 2 индивидуальные консультации для руководителя (по 50 мин)

**Формат:** Онлайн или офлайн (Москва)
**Срок:** 2 недели

---

### Пакет «Премиум» (для крупных команд 50+ человек)
**Цена:** 120,000₽
**Что входит:**
1. Комплексный аудит (опросы, интервью, анализ текучки)
2. Стратегический отчёт с планом действий на 6 месяцев
3. 5 групповых сессий для разных уровней (руководители, middle, junior)
4. 5 индивидуальных консультаций для сотрудников (по их выбору)
5. Сопровождение HR в течение месяца (4 созвона)
6. Презентация результатов для топ-менеджмента

**Формат:** Онлайн + офлайн
**Срок:** 1 месяц

## Как это продаётся

### Через Lead-Gen (быстрый win)
Форма в Яндексе: «Бесплатный аудит усталости команды ИБ»
→ Встреча-знакомство (30 мин)
→ Предложение подходящего пакета

### Через Telegram-рассылку
Дайджесты с полезностями → доверие
→ Предложение диагностики
→ Пакет

### Через личный нетворкинг
Выступления на конференциях (SecurIT, PHDays)
→ Контакты HR
→ Презентация пакета
