## Рекомендации по генерации

**Для Midjourney:**
- Добавьте `--ar 1:1` для квадратного формата
- Добавьте `--style raw` для более чистого результата
- Используйте `--v 6` для последней версии

**Для DALL-E 3:**
- Формат 1:1 (1024x1024) оптимален
- Добавьте "professional, clean design, suitable for Telegram channel"

**Для Kandinsky:**
- Стиль: "Дизайн" или "Реклама"
- Соотношение 1:1

## Цветовая палитра канала

**Основные:**
- Зелёный: `#2D5A4A` (primary)
- Кремовый: `#F5F1EB` (background)
- Тёмно-зелёный: `#1E3D32` (text)

**Акценты:**
- Оранжевый: для предупреждений/алертов
- Красный: только для критических ошибок
- Золотой: для бонусов/подарков

**Шрифты:**
- Заголовки: Montserrat Bold
- Текст: Montserrat Regular
- Декоративный: Source Serif Pro (для цитат)
