# 🤖 Структура бота @ShchitOtMoshennikov_bot

## Назначение
Бот для приёма заявок клиентов психолога Камилы Золотовой (SecureMind).

## 🎯 Основные функции

### 1. Приём заявок с сайта
- Форма на сайте отправляет данные в бот
- Автоматическое уведомление психолога
- Сохранение контактов клиента

### 2. Интерактивное меню (/start)
```
📝 Записаться на консультацию
❓ Частые вопросы (FAQ)
📚 Бесплатные материалы
📞 Связаться с психологом
```

### 3. Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/consultation` | Запись на консультацию |
| `/faq` | Частые вопросы |
| `/materials` | Полезные материалы |
| `/contact` | Контакты |
| `/price` | Стоимость услуг |
| `/about` | О психологе |

## 📱 Как использовать

### Для клиентов:
1. Пишут /start → видят меню
2. Выбирают нужный раздел
3. Заполняют заявку или получают материалы

### Для психолога (Камила):
1. Получает уведомления о новых заявках
2. Может отвечать клиентам через бот
3. Может отправлять материалы через команды

## 🚀 Команды для отправки материалов

Отправить чек-лист клиенту:
```
/send_checklist [user_id] anxiety
```

Отправить презентацию:
```
/send_presentation [user_id] burnout
```
